<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Install flutter
* bash
  ```sh
   git clone https://github.com/flutter/flutter.git -b stable
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/raysummee/dino-runner.git
   ```
2. Install pub plugins
   ```sh
   flutter pub get
   ```
3. Run the app
   ```sh
   flutter run
   ```
